<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>AI Recommended Prompts</strong> API returns structured AI analysis for the <strong>Bithumb KRW</strong> crypto
  market for a given <code>type</code> (signal type). Use it to quickly understand market conditions, asset movements,
  capital flows, and emerging narratives.
</p>

<Note>
  Supported <code>type</code> values are defined in the OpenAPI enum and may expand over time. Clients should handle
  unknown values gracefully.
</Note>

<h3 style={{ marginTop: 20 }}>Available types</h3>

<ul style={{ fontSize: 15, lineHeight: 1.8 }}>
  <li><code>undervalued_asset</code>: potentially undervalued assets</li>
  <li><code>sector_leaders</code>: leading sectors / sector leaders</li>
  <li><code>bullish_assets</code>: bullish assets</li>
  <li><code>stablecoin_inflows</code>: stablecoin inflows</li>
  <li><code>asset_price_surge</code>: notable price surges</li>
  <li><code>asset_price_decline</code>: notable price declines</li>
  <li><code>positive_news_top5</code>: top positive news</li>
  <li><code>sentiment_divergence</code>: sentiment divergence signals</li>
  <li><code>trending_issues</code>: trending market issues</li>
  <li><code>golden_cross</code>: golden-cross technical signals</li>
  <li><code>exchange_outflows</code>: exchange outflows</li>
  <li><code>top_active_chain</code>: most active chains</li>
  <li><code>hottest_meme_coins</code>: hottest meme coins</li>
  <li><code>btc_etf_flow</code>: BTC ETF flows</li>
  <li><code>kol_sectors</code>: KOL sector discussions</li>
  <li><code>kol_themes</code>: KOL theme narratives</li>
</ul>

<h3 style={{ marginTop: 20 }}>How to use</h3>

<Steps>
  <Step title="Choose a type">
    Set <code>type</code> (required) as the signal type. Different types produce different structured outputs.
  </Step>

  <Step title="Optional: choose a date">
    Provide <code>date</code> (YYYY-MM-DD) to view historical context. Default is the latest available snapshot.
  </Step>
</Steps>

<Warning>
  Outputs are for informational purposes only and should not be interpreted as financial advice.
</Warning>
