<h2 style={{ fontSize: "22px", marginBottom: 8 }}>Overview</h2>

<p style={{ fontSize: 15, lineHeight: 1.8 }}>
  The <strong>AI Recommended Prompts</strong> API returns structured AI analysis for the <strong>Bithumb KRW</strong> crypto
  market for a given <code>type</code> (signal type). Use it to quickly understand market conditions, asset movements,
  capital flows, and emerging narratives.
</p>

<Note>
  Supported <code>type</code> values are defined in the OpenAPI enum and may expand over time. Clients should handle
  unknown values gracefully.
</Note>

<h3 style={{ marginTop: 20 }}>How to use</h3>

<Steps>
  <Step title="Choose a type">
    Set <code>type</code> (required) as the signal type. Different types produce different structured outputs.
  </Step>

  <Step title="Ticker when needed">
    When <code>type</code> is <code>asset_price_surge</code> or <code>asset_price_decline</code>, provide <code>ticker</code> (required for that type).
  </Step>

  <Step title="Optional: choose a date">
    Provide <code>date</code> (YYYY-MM-DD) to view historical context. Default is the latest available snapshot.
  </Step>
</Steps>

<Warning>
  Outputs are for informational purposes only and should not be interpreted as financial advice.
</Warning>
